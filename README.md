This mod requires Steamodded. I'd recommend using release version [1.0.0-beta-1620a](https://github.com/Steamodded/smods/releases/tag/1.0.0-beta-1620a) or later.

Use WASD or the arrow keys to move the indicator and drop balls. These controls can be changed in the config.

Some vanilla cards and features are not present. In addition, to avoid crashes, please use few other mods and avoid using content mods (feel free to test the limits). JokerDisplay is currently not compatible, apologies!  
If you are interested in playing Multiplayer, please use version [0.4.2](https://github.com/Balatro-Multiplayer/BalatroMultiplayer/releases/tag/v0.4.2) or later, and **TURN OFF SCORE PREVIEW**! For the least unstable experience, please use either the Standard, Traditional, or Vanilla ruleset.

If you have any queries, idea suggestions, or bug/crash notices, feel free to reach out to me via the [discord discussion thread](https://discord.com/channels/1116389027176787968/1439235829229813834)!
