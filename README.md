This mod requires Steamodded. I'd recommend using release version [1.0.0-beta-1016c](https://github.com/Steamodded/smods/releases/tag/1.0.0-beta-1016c) or later.

Use WASD or the arrow keys to move the indicator and drop balls.

Many vanilla cards and features are not present. In addition, to avoid crashes, please use few other mods and avoid using content mods (feel free to test the limits).

If you have any queries, idea suggestions, or bug/crash notices, feel free to reach out to me via the [discord discussion thread](https://discord.com/channels/1116389027176787968/1439235829229813834)!
